# Social Link Profile - Frontend Mentor Challenge

This is a solution to the **Social Link Profile** challenge from [Frontend Mentor](https://www.frontendmentor.io/). The project is a profile card layout that displays personal information and links to social media profiles, designed to be responsive across devices.

## Project Overview

This project showcases a simple profile card with the following features:
- **Profile Picture**: A circular avatar.
- **User Info**: Name, location, and description.
- **Social Links**: 5 social media links (GitHub, LinkedIn, Twitter, Instagram, Frontend Mentor).
- **Responsive Design**: The layout adapts for desktop, tablet, and mobile using CSS Grid and Media Queries.


## Technologies Used

- HTML
- CSS (Flexbox, Grid, Media Queries)
- Google Fonts (`Inter`)
